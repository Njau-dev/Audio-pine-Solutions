## 🚀 Getting Started

### 📋 Prerequisites

- Node.js ≥16.x
- npm ≥8.x
- Git

### ⚙️ Installation

```bash
# 1. Clone repository
git clone https://github.com/Njau-dev/Audio-pine-Solutions

# 2. Install dependencies
npm install

# 3. run server
npm run dev
```
